No livro PHP: a Bíblia, Tim Converse explica o que é PHP.

O PHP é uma linguagem para criação de scripts para a web do lado servidor embutidos em HTML, cujo código-font é aberto, e é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de códigos em páginas HTML normais - código esse que é interpretado á medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas páginas Web com o banco de dados do lado servidor.
