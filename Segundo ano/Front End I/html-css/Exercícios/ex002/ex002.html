Parágrafos e quebras linha

IMPORTANTE: Visualize sempre o código fonte da página

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p>

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>

OBS1: a tag <br> não possui fechamento com <br/>

OBS2: JAMAIS use vários <br> <br> <br> ... para criar um espaço grande na sua página. Mais tarde veremos como fazer isso usando CSS

Ao adicionarmos a letra "&" antes do código correto, teremos as HTML entities

It: Exibe o símbolo <(Less Than)
gt; Exibe símbolos >(Greater Than)
amp: Exibe o símbolo & (Ampersand)
copy: Exibe o símbolo © (Copyright)
reg: Exibe o símbolo ® (Registered Trademark)
euro: Exibe o símbolo € (Euro)
pound: Exibe o símbolo £ (Pound Sterling)
